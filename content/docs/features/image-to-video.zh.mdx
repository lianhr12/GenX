## 让你的图片活起来

图生视频是 GenX 的核心功能，让你能够将静态图片转换为具有艺术风格的动态动画视频。

## 工作原理

1. **上传图片** - 添加 1-10 张图片
2. **选择风格** - 选择艺术风格
3. **配置设置** - 设置时长和运动
4. **生成** - AI 创建你的视频
5. **下载** - 获取你的杰作

## 输入要求

| 参数 | 规格 |
|-----|-----|
| 格式 | JPG、PNG |
| 最大文件大小 | 每张 10MB |
| 图片数量 | 1-10 张 |
| 推荐分辨率 | 1080p 或更高 |

## 单图动画

当你上传单张图片时，GenX 将：
- 应用你选择的艺术风格
- 添加自然、流动的运动
- 创建深度和层次
- 生成 3-30 秒的视频

**最适合：**
- 艺术转换
- 社交媒体内容
- 产品动画
- 人像效果

## 多图序列

当你上传 2-10 张图片时，GenX 将：
- 在图片之间创建平滑过渡
- 保持整体风格一致
- 应用自然的流程和节奏
- 生成连贯的视频叙事

**最适合：**
- 照片蒙太奇
- 活动精彩集锦
- 前后对比展示
- 故事序列

## 配置选项

### 艺术风格

从五种风格中选择：
- **赛博朋克** - 未来主义、霓虹美学
- **水彩画** - 柔和、流动的艺术效果
- **油画** - 丰富、有质感的古典风格
- **动漫** - 日本动画风格
- **流体艺术** - 抽象液体运动

### 视频时长

| 时长 | 最适合 |
|-----|--------|
| 3-5 秒 | 社交媒体、快速展示 |
| 6-15 秒 | 标准内容、演示 |
| 16-30 秒 | 较长叙事、详细动画 |

### 运动强度

- **低** - 微妙、温和的运动
- **中** - 平衡的动画（推荐）
- **高** - 动感、充满活力的运动

## 最佳实践

### 图像质量

高质量输入产生更好的结果：
- 使用光线充足的图像
- 避免严重压缩
- 包含清晰的主体
- 多张图片保持一致的宽高比

### 风格匹配

将图片与适当的风格匹配：

| 图片类型 | 推荐风格 |
|---------|---------|
| 人像 | 油画、动漫 |
| 风景 | 水彩画、油画 |
| 城市场景 | 赛博朋克 |
| 抽象 | 流体艺术 |
| 角色 | 动漫 |

### 多图技巧

为了最佳序列效果：
- 使用相似的构图
- 保持色彩一致性
- 按逻辑顺序排列图片
- 考虑图片之间的视觉流程

## 积分使用

| 时长 | 积分 |
|-----|------|
| 3-5 秒 | 5 积分 |
| 6-15 秒 | 10 积分 |
| 16-30 秒 | 20 积分 |
