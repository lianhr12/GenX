## Writing Effective Prompts

The key to great results is writing clear, descriptive prompts.

### Prompt Structure

A good prompt includes:

| Element | Example |
|---------|---------|
| Subject | "A lone wolf" |
| Action | "standing on a cliff" |
| Environment | "overlooking a misty valley" |
| Atmosphere | "at dawn, golden light" |

### Complete Example

> "A lone wolf standing on a cliff overlooking a misty valley at dawn, golden light illuminating its fur, majestic and peaceful atmosphere"

## Prompt Tips

### Do:
- Be specific about visual elements
- Describe colors and lighting
- Include mood and atmosphere
- Mention movement when relevant

### Don't:
- Use abstract concepts that can't be visualized
- Write overly long, complex prompts
- Include contradictory descriptions
- Expect exact literal interpretation

## Prompt Templates

### Nature Scenes
```
[Subject] in [environment], [time of day], [weather/atmosphere]
```
Example: "A deer drinking from a stream in a misty forest, early morning, golden sunlight filtering through trees"

### Urban Scenes
```
[Subject] in [city setting], [time], [mood], [weather]
```
Example: "A busy intersection in Tokyo at night, rain falling, neon lights reflecting everywhere"

### Characters
```
[Character description] [action], [setting], [mood]
```
Example: "A young artist painting at an easel, sunlit studio, focused and inspired"

### Abstract
```
[Colors/shapes] [movement], [emotion], [texture]
```
Example: "Swirling blues and golds, cosmic energy, peaceful and meditative"

## Configuration Options

### Duration
- **3-5 seconds** - Quick concepts
- **6-15 seconds** - Standard content
- **16-30 seconds** - Detailed scenes

### Motion Intensity
- **Low** - Subtle, calm scenes
- **Medium** - Balanced (recommended)
- **High** - Dynamic, energetic content

## Troubleshooting

### Output doesn't match prompt
- Simplify your description
- Focus on 2-3 key visual elements
- Be more specific about details

### Video looks generic
- Add unique, specific details
- Specify colors and lighting
- Include mood descriptors

### Unexpected elements appear
- Your prompt may be ambiguous
- Try rephrasing with clearer language
- Remove conflicting descriptions
