## 从想象力创作

文生视频让你能够纯粹从文字描述生成艺术视频。不需要图片——只需描述你的愿景，让 AI 将其变为现实。

## 工作原理

1. **撰写提示词** - 描述你想看到的内容
2. **选择风格** - 选择艺术风格
3. **配置设置** - 设置时长和运动
4. **生成** - AI 创建你的愿景
5. **下载** - 获取你的视频

## 撰写有效的提示词

获得出色结果的关键是写出清晰、描述性的提示词。

### 提示词结构

一个好的提示词包括：

| 元素 | 示例 |
|-----|-----|
| 主体 | "一只孤独的狼" |
| 动作 | "站在悬崖上" |
| 环境 | "俯瞰薄雾笼罩的山谷" |
| 氛围 | "黎明时分，金色光芒" |

### 完整示例

> "一只孤独的狼站在悬崖上俯瞰薄雾笼罩的山谷，黎明时分，金色光芒照亮它的皮毛，雄伟而平和的氛围"

## 提示词技巧

### 应该：
- 具体描述视觉元素
- 描述颜色和光线
- 包含氛围和情绪
- 在相关时提及运动

### 不应该：
- 使用无法可视化的抽象概念
- 写过长、复杂的提示词
- 包含矛盾的描述
- 期望完全字面的解释

## 提示词模板

### 自然场景
```
[主体] 在 [环境] 中，[时间]，[天气/氛围]
```
示例："一只鹿在薄雾森林中的溪流边饮水，清晨，金色阳光透过树木过滤"

### 城市场景
```
[主体] 在 [城市环境] 中，[时间]，[氛围]，[天气]
```
示例："夜晚东京繁忙的十字路口，雨水落下，霓虹灯到处反射"

### 角色
```
[角色描述] [动作]，[环境]，[氛围]
```
示例："一位年轻艺术家在画架前绑画，阳光充足的工作室，专注而充满灵感"

### 抽象
```
[颜色/形状] [运动]，[情感]，[质感]
```
示例："旋转的蓝色和金色，宇宙能量，平静和冥想"

## 风格推荐

| 提示词类型 | 最佳风格 |
|-----------|---------|
| 未来/科技 | 赛博朋克 |
| 自然/平静 | 水彩画、油画 |
| 角色 | 动漫、油画 |
| 抽象 | 流体艺术 |
| 动作/动感 | 动漫、赛博朋克 |

## 配置选项

### 时长
- **3-5 秒** - 快速概念
- **6-15 秒** - 标准内容
- **16-30 秒** - 详细场景

### 运动强度
- **低** - 微妙、平静的场景
- **中** - 平衡（推荐）
- **高** - 动感、充满活力的内容

## 积分使用

| 时长 | 积分 |
|-----|------|
| 3-5 秒 | 5 积分 |
| 6-15 秒 | 10 积分 |
| 16-30 秒 | 20 积分 |

## 故障排除

### 输出与提示词不匹配
- 简化你的描述
- 专注于 2-3 个关键视觉元素
- 更具体地描述细节

### 视频看起来很普通
- 添加独特、具体的细节
- 指定颜色和光线
- 包含氛围描述词

### 出现意外元素
- 你的提示词可能有歧义
- 尝试用更清晰的语言重新表述
- 删除矛盾的描述
