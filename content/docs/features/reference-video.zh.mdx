## 创作一致的内容

参考视频允许你使用现有视频作为风格指南，确保内容的美学一致性。

## 工作原理

1. **上传参考** - 添加代表你想要风格的视频
2. **提供源内容** - 添加新视频的图片或文本
3. **调整影响程度** - 设置参考对输出的影响强度
4. **生成** - AI 创建风格化内容
5. **下载** - 获取你的一致性视频

## 参考视频捕捉什么

AI 分析你的参考以理解：

- **视觉风格** - 颜色、光线、对比度
- **运动模式** - 移动速度、流程、节奏
- **构图** - 取景、焦点区域
- **美学感觉** - 整体氛围和情绪

## 输入方式

### 参考 + 图片

上传参考视频以及 1-10 张图片。图片将被转换以匹配参考风格。

**最适合：**
- 品牌一致的产品视频
- 将风格迁移到新内容
- 视觉系列创作

### 参考 + 文本

上传参考视频并提供文本提示。将生成与参考美学匹配的新内容。

**最适合：**
- 扩展现有视觉主题
- 创建风格变体
- 构建连贯的内容库

## 输入要求

### 参考视频

| 参数 | 规格 |
|-----|-----|
| 格式 | MP4、MOV、WebM |
| 最大文件大小 | 100MB |
| 时长 | 2-60 秒 |
| 分辨率 | 480p - 4K |

### 源图片（如果使用）

| 参数 | 规格 |
|-----|-----|
| 格式 | JPG、PNG |
| 最大文件大小 | 每张 10MB |
| 数量 | 1-10 张 |

## 风格影响设置

控制参考对输出的影响强度：

| 级别 | 效果 | 最适合 |
|-----|-----|--------|
| 低（20-40%） | 微妙提示 | 添加轻微风格触感 |
| 中（40-60%） | 平衡混合 | 一般用途 |
| 高（60-80%） | 强匹配 | 品牌一致性 |
| 最大（80-100%） | 接近复制 | 精确风格匹配 |

## 最佳实践

### 选择好的参考

**理想的参考视频：**
- 始终一致的风格
- 清晰的视觉元素
- 高质量的素材
- 代表你想要的美学效果

**避免：**
- 具有多种冲突风格的视频
- 质量差或严重压缩
- 非常快或混乱的运动
- 过于复杂的构图

### 建立参考库

为以下创建参考视频集合：
- 品牌视觉身份
- 不同的氛围/季节
- 各种内容类型
- 特定平台

## 积分使用

由于需要额外处理，参考视频使用的积分稍多：

| 时长 | 积分 |
|-----|------|
| 3-5 秒 | 8 积分 |
| 6-15 秒 | 15 积分 |
| 16-30 秒 | 25 积分 |

## 用例

### 品牌一致性
在所有营销视频中保持相同的视觉风格，确保品牌识别度。

### 系列创作
使用一个视频作为风格主控，创建具有一致美学的系列内容。

### 风格复制
捕捉并将特定视觉风格应用到新内容，无需手动重新创建。

### 内容扩展
生成与现有成功视频匹配的更多内容。
