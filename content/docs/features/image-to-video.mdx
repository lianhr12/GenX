## Bring Your Images to Life

Image to Video is GenX's core feature, allowing you to transform static images into dynamic, animated videos with artistic styling.

## How It Works

1. **Upload Images** - Add 1-10 images
2. **Choose Style** - Select an artistic style
3. **Configure Settings** - Set duration and motion
4. **Generate** - AI creates your video
5. **Download** - Get your masterpiece

## Input Requirements

| Parameter | Specification |
|-----------|--------------|
| Formats | JPG, PNG |
| Max file size | 10MB per image |
| Image count | 1-10 images |
| Recommended resolution | 1080p or higher |

## Single Image Animation

When you upload a single image, GenX will:
- Apply your chosen artistic style
- Add natural, flowing motion
- Create depth and dimension
- Generate a 3-30 second video

**Best for:**
- Artistic transformations
- Social media content
- Product animations
- Portrait effects

## Multiple Image Sequences

When you upload 2-10 images, GenX will:
- Create smooth transitions between images
- Maintain consistent styling throughout
- Apply natural flow and pacing
- Generate a cohesive video narrative

**Best for:**
- Photo montages
- Event highlights
- Before/after reveals
- Story sequences

## Configuration Options

### Art Style

Choose from five styles:
- **Cyberpunk** - Futuristic, neon aesthetics
- **Watercolor** - Soft, flowing artistic effects
- **Oil Painting** - Rich, textured classical feel
- **Anime** - Japanese animation style
- **Fluid Art** - Abstract liquid motion

### Video Duration

| Duration | Best For |
|----------|----------|
| 3-5 seconds | Social media, quick reveals |
| 6-15 seconds | Standard content, presentations |
| 16-30 seconds | Longer narratives, detailed animations |

### Motion Intensity

- **Low** - Subtle, gentle movement
- **Medium** - Balanced animation (recommended)
- **High** - Dynamic, energetic motion

## Best Practices

### Image Quality

High-quality inputs produce better results:
- Use well-lit images
- Avoid heavy compression
- Include clear subjects
- Maintain consistent aspect ratios for multiple images

### Style Matching

Match your images to appropriate styles:

| Image Type | Recommended Styles |
|-----------|-------------------|
| Portraits | Oil Painting, Anime |
| Landscapes | Watercolor, Oil Painting |
| Urban scenes | Cyberpunk |
| Abstract | Fluid Art |
| Characters | Anime |

### Multiple Image Tips

For best sequences:
- Use similar compositions
- Maintain color consistency
- Order images logically
- Consider visual flow between images
