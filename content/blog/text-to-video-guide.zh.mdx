## 从想象力创作视频

文生视频是 GenX 最强大的功能之一，让你能够纯粹从文字描述生成艺术视频。不需要图片——只需要你的想象力和文字。

## 文生视频如何工作

GenX 的 AI 分析你的文本提示以理解：
- 你描述的场景
- 氛围和环境
- 关键视觉元素
- 运动和动作

然后它生成一个独特的视频，以你选择的艺术风格将你的描述变为现实。

## 撰写有效的提示词

获得出色文生视频效果的关键是写出清晰、描述性的提示词。以下是如何撰写完美提示词的方法：

### 基本提示词结构

一个好的提示词通常包括：

1. **主体** - 主要焦点是什么？
2. **动作** - 正在发生什么？
3. **环境** - 这发生在哪里？
4. **氛围/风格** - 应该唤起什么感觉？

### 示例提示词

**简单：**
> 一只蝴蝶落在花朵上

**详细：**
> 一只鲜艳的帝王蝶轻轻落在阳光普照的草地上的紫色薰衣草花上，柔和的晨光透过

**氛围感：**
> 一个孤独的身影走在夜晚被雨水浸湿的城市街道上，霓虹灯标志倒映在湿漉漉的路面上，神秘而电影感的氛围

### 提示词撰写技巧

**应该：**
- 具体说明颜色、光线和氛围
- 描述运动和动作
- 包含情感或氛围描述词
- 如果相关，提及时间或天气

**避免：**
- 模糊或抽象的概念
- 太多主体竞争注意力
- 矛盾的描述
- 过长、复杂的提示词

## 分步指南

### 第一步：导航到文生视频

从仪表板进入 [创作 → 文生视频](/create/video)。

### 第二步：撰写提示词

在文本字段中输入你的描述。记住：
- 描述性但清晰
- 专注于视觉元素
- 考虑你将选择的风格

### 第三步：选择艺术风格

从以下选择：
- **赛博朋克** - 最适合未来主义、科技主题的提示词
- **水彩画** - 非常适合自然、柔和的场景
- **油画** - 非常适合戏剧性、艺术性的场景
- **动漫** - 非常适合以角色为中心的内容
- **流体艺术** - 最适合抽象、流动的概念

### 第四步：配置设置

**时长：** 3-30 秒
**运动：** 根据提示词调整强度

### 第五步：生成并查看

点击生成并等待视频完成。查看并根据需要调整提示词重新生成。

## 提示词模板

### 自然场景
```
[动物] [动作] 在 [环境] 中，[时间]，[天气/氛围]
```
示例："一只鹿在薄雾森林中的清澈溪流边饮水，清晨，金色阳光透过树木过滤"

### 城市场景
```
[主体] 在 [城市/城市环境] 中，[时间]，[氛围]，[天气]
```
示例："夜晚东京繁忙的十字路口，雨水落下，霓虹灯到处反射，繁华的能量"

### 抽象概念
```
[颜色/形状] [运动]，[情感]，[质感]
```
示例："旋转的蓝色和紫色，宇宙能量，平静和冥想，丝绸般的质感"

### 角色场景
```
[角色描述] [动作]，[环境]，[氛围/情感]
```
示例："一位年轻艺术家在画架前绑画，阳光充足的工作室里满是画布，专注而充满灵感"

## 风格 + 提示词组合

| 提示词类型 | 最佳风格 |
|-----------|---------|
| 未来城市 | 赛博朋克 |
| 自然、风景 | 水彩画、油画 |
| 角色、人物 | 动漫、油画 |
| 抽象概念 | 流体艺术、水彩画 |
| 动作场景 | 动漫、赛博朋克 |

## 高级技巧

### 组合元素
叠加多个视觉元素以获得更丰富的场景：
> "樱花围绕传统日本寺庙飘落，下方池塘中可见锦鲤，柔和的粉色和橙色夕阳天空"

### 情感引导
添加情感指导以影响输出：
> "雪中舒适的小屋，窗户里温暖的灯光，平静而温馨，圣诞氛围"

### 镜头运动提示
建议视角：
> "秋天森林中蜿蜒河流的鸟瞰图，慢慢展现隐藏的瀑布"

## 故障排除

**输出与提示词不匹配：**
- 简化你的描述
- 专注于 2-3 个关键元素
- 更具体地描述视觉细节

**视频看起来很普通：**
- 添加独特的细节
- 指定颜色和光线
- 包含氛围描述词

**运动感觉不对：**
- 明确描述运动类型
- 调整运动强度设置
- 先尝试更短的时长

## 积分使用

文生视频使用与图生视频相同的积分结构：

| 时长 | 积分 |
|-----|------|
| 3-5 秒 | 5 积分 |
| 6-15 秒 | 10 积分 |
| 16-30 秒 | 20 积分 |
