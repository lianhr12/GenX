## Facebook 数据删除说明

本页面说明如何删除您的 GenX.art 账户以及我们从 Facebook 账户获取的任何数据。

### 我们从 Facebook 收集的数据

当您使用 Facebook 登录 GenX.art 时，我们会从 Facebook 接收以下信息：

- **姓名**：您在 Facebook 上显示的全名
- **邮箱地址**：与您的 Facebook 账户关联的主要邮箱地址
- **头像**：您的 Facebook 头像（用作您的个人资料头像）
- **Facebook 用户 ID**：您 Facebook 账户的唯一标识符

### 如何删除您的账户和数据

您有两种选择来删除您的账户和数据：

#### 选项 1：通过 GenX.art 设置删除账户

1. **登录 GenX.art**，使用您的 Facebook 账户
2. **导航到设置**：
   - 点击右上角的头像
   - 从下拉菜单中选择"设置"
3. **进入安全设置**：
   - 在左侧边栏中，点击"安全"
   - 向下滚动到"删除账户"部分
4. **删除您的账户**：
   - 点击"删除账户"按钮
   - 在弹出对话框中确认您的决定
   - 您的账户和所有相关数据将在 30 天内被永久删除

#### 选项 2：通过邮件请求删除

如果您无法访问您的账户或希望通过邮件请求删除：

1. **向我们发送邮件**至 [support@genx.art](mailto:support@genx.art)
2. **包含以下信息**：
   - 您的 Facebook 邮箱地址
   - 您的 Facebook 姓名
   - 明确的账户删除请求声明
3. **验证流程**：
   - 我们可能要求额外验证以确认您的身份
   - 验证后，我们将在 30 天内处理您的删除请求

### 删除后会发生什么

当您删除账户时：

- **立即生效**：
  - 您将被登出所有活动会话
  - 您的账户将被标记为待删除
  - 您将无法访问您的账户或数据

- **30 天内**：
  - 您的所有个人数据将从我们的服务器中永久删除
  - 您生成的视频和上传的图片将被删除
  - 您的支付历史将被匿名化（仅为法律合规而保留）
  - 我们收集的所有 Facebook 数据将被删除

- **我们无法删除的数据**：
  - 不再识别您的匿名使用数据
  - 税务和法律合规所需的支付记录（保留 7 年）
  - 可能包含您的 IP 地址的系统日志（出于安全原因保留 90 天）

### Facebook 的数据删除工具

您也可以使用 Facebook 的数据删除工具：

1. **进入 Facebook 设置**：
   - 点击 Facebook 右上角的向下箭头
   - 选择"设置与隐私" > "设置"
   
2. **应用和网站**：
   - 在左侧菜单中点击"应用和网站"
   - 在活跃应用列表中找到"GenX.art"
   
3. **移除应用**：
   - 点击 GenX.art 旁边的"移除"
   - 勾选"删除 GenX.art 代表您发布的帖子、照片或视频"复选框
   - 点击"移除"

**注意**：通过 Facebook 移除应用将撤销我们对您 Facebook 数据的访问权限，但不会自动删除您的 GenX.art 账户。您必须按照上述账户删除步骤来完全删除您的数据。

### 联系我们

如果您对数据删除有任何疑问或需要删除账户方面的帮助，请联系我们：

- **邮箱**：[support@genx.art](mailto:support@genx.art)
- **邮件主题**："数据删除请求"

我们将根据适用的数据保护法律要求，在 30 天内回复您的请求。
