## 一、Executive Summary 执行摘要

InVideo AI 是一个 **AI 视频创作平台**，定位为"Create Videos Without Limits"（无限制创建视频）。其 Landing Page 采用简洁的白色主题设计，以内容展示墙和 AI 编辑能力演示为核心，突出强调"创意无限"的品牌理念。

### 核心亮点

- **定位清晰**: 专注 AI 视频生成，而非图像+视频的综合平台
- **内容驱动**: 以真实用户作品/模板为核心展示
- **AI 编辑**: 强调"Edit like you think"的自然语言编辑能力
- **双产品线**: invideo AI (AI生成) + invideo Studio (传统编辑)

### 关键数据

| 指标       | 数值                                                 |
| ---------- | ---------------------------------------------------- |
| 页面标题   | "Invideo AI - Create videos without limits" (47字符) |
| H1 标题    | "Create Videos Without Limits"                       |
| 主要 CTA   | Try invideo / Sign up                                |
| 页面总元素 | 448 refs                                             |
| 交互元素   | 275 refs                                             |
| 用户规模   | 30M+ users (声称)                                    |

**优点**:

- ✅ 极简设计，白色背景突出内容
- ✅ H1 字体巨大、粗体，冲击力强
- ✅ "VIDEOS" 文字内嵌视频纹理效果，视觉创新
- ✅ AI Prompt 输入框作为主 CTA，降低使用门槛

**改进点**:

- ⚠️ 首屏没有展示产品截图或视频预览

## 六、竞争优势分析

### 6.1 与 Pollo AI 对比

| 维度         | InVideo AI                      | Pollo AI                |
| ------------ | ------------------------------- | ----------------------- |
| **定位**     | 专注视频创作                    | 图像+视频综合平台       |
| **设计风格** | 白色简约                        | 深色创意                |
| **核心亮点** | 内容展示墙 + AI 编辑            | 模型聚合 + 特效库       |
| **模型策略** | 集成多模型但不强调              | 强调"ALL Models in ONE" |
| **用户证言** | 5个行业案例                     | 用户数量                |
| **产品线**   | 双产品线 (AI + Studio)          | 单一 AI 平台            |
| **特效分类** | 场景化命名 (Million Dollar Ads) | 技术化命名 (AI Kissing) |

### 6.2 InVideo 核心竞争力

1. **内容驱动营销**: 以真实模板/作品吸引用户
2. **场景化命名**: "Million Dollar Ads"、"Money Shot" 激发商业想象
3. **双产品线**: AI 生成 + 传统编辑覆盖不同需求
4. **自然语言编辑**: "Edit like you think" 强调易用性
5. **社区生态**: 展示用户作品，形成创作者社区
6. **行业证言**: 不同行业用户背书增加可信度

### 6.3 潜在劣势

1. **首屏信息量**: 首屏没有展示产品截图，依赖用户滚动
2. **定价不透明**: 主要定价信息需跳转页面
3. **模型品牌弱化**: 不如 Pollo AI 强调模型聚合优势
4. **图像能力**: 定位视频为主，图像生成相对弱化
